# 🎵 Spotlight

Spotlight é uma interface inspirada em plataformas de streaming de música, desenvolvida com foco em HTML semântico, CSS moderno e responsividade. O projeto foi criado como prática de desenvolvimento front-end, priorizando organização de código, acessibilidade e uma experiência visual consistente em diferentes tamanhos de tela.

## 🚀 Acesse o projeto

🔗 **GitHub Pages:** https://viniboy404.github.io/spotlight/

## ✨ Funcionalidades

- Interface moderna inspirada em serviços de streaming
- Layout totalmente responsivo
- Hero section personalizada
- Barra lateral de navegação
- Seções de playlists
- Player de música fixo
- Animações e transições suaves
- Código semântico e acessível

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Variáveis CSS
- Media Queries

## ♿ Acessibilidade

O projeto foi desenvolvido seguindo boas práticas de acessibilidade, incluindo:

- Estrutura HTML semântica
- Navegação por teclado
- Estados de foco (`:focus-visible`)
- Suporte a `prefers-reduced-motion`
- Uso adequado de atributos ARIA

## 📱 Responsividade

A interface foi adaptada para diferentes dispositivos utilizando breakpoints específicos para:

- Desktop
- Tablet
- Smartphone
- Smartphones compactos
